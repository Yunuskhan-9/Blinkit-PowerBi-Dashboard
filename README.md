# Blinkit-PowerBi-Dashboard
Blinkit Power BI Analysis
This project showcases an interactive Power BI dashboard built on Blinkit’s dataset, focusing on product performance, sales trends, and customer behavior. The analysis provides actionable insights into pricing strategies, outlet performance, and demand patterns across different categories.
🔑 Features
- Product category-wise sales and demand analysis
- Price vs. sales trends to evaluate profitability
- Outlet performance comparison across locations
- Customer preference insights with interactive filters
- KPIs and visuals for quick decision-making
🎯 Outcome
The dashboard transforms Blinkit’s raw dataset into business intelligence, helping identify growth opportunities, optimize pricing, and improve outlet efficiency in the quick-commerce industry.
